## [0.1.0] - 2025-10-26

### Added

- **Initial Release**: Project setup
  - SvelteKit 5 with TypeScript
  - Drizzle ORM with Turso database
  - Lucia v3 authentication framework
  - Tailwind CSS 4 for styling
  - Vitest for unit testing
  - Playwright for E2E testing
  - Storybook for component development
  - Paraglide for i18n (English & Indonesian)
  - ESLint & Prettier for code quality
  - MCP integration for AI assistants

### Documentation

- Initial project structure
- README.md with setup instructions
- PROGRESS.md for tracking development
- ROADMAP.md with development timeline
- Concept documents for future features
