# Quick Team Guide: Kebutuhan Tim untuk Platform Ini

**Created**: 2025-10-27  
**Quick Reference**: Untuk yang ingin tahu skill apa saja yang dibutuhkan

## 🎯 Ringkasan: 10 Profesi Core yang Dibutuhkan

### Untuk Build Platform (Initial MVP):

1. **Frontend Developer**
   - Svelte 5, TypeScript, Tailwind CSS
   - Build UI yang intuitif

2. **Backend Developer**
   - Node.js, PostgreSQL, Drizzle ORM
   - Database & API

3. **DevOps Engineer**
   - Docker, CI/CD, Cloud
   - Setup production infrastructure

4. **UI/UX Designer**
   - Figma, Component System
   - Design yang "ringan di mata"

5. **QA/Tester**
   - Vitest, Playwright
   - Ensure quality

6. **Technical Writer**
   - Documentation
   - Tutorials & guides

7. **Community Manager**
   - Build community
   - User support

8. **Security Specialist**
   - Security audit
   - Vulnerability prevention

9. **Product Manager**
   - Roadmap & strategy
   - Feature prioritization

10. **Database Architect**
    - Schema design
    - Performance optimization

## 💡 Yang Perlu Diketahui

**Tech Stack:**
- Frontend: SvelteKit 5, Svelte 5, Tailwind CSS 4
- Backend: Node.js, TypeScript, PostgreSQL
- ORM: Drizzle
- Auth: Lucia v3
- Testing: Vitest, Playwright
- Deployment: Docker, Cloudflare

**Filosofi Platform:**
- "Ringan di mata" - Simple di luar, kompleks di dalam
- Open source core (MIT License)
- Self-hostable
- Kedaulatan data

**Mission:**
- Bagian dari ekosistem open source Indonesia
- Masuk dalam hirarki: SLiMS, OpenSID, 2USE, We Will Shine
- Posisi: HR & Manajerial Perusahaan yang Akuntable
